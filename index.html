Current Projects

Hisha Choose Your Own Adventure Game

A choose your own adventure game powered by an LLM on the backend. A pillar of this project is to have the entire system run locally.

Currently support is limited to a locally running Ollama server but with that support comes inherent support for a big selection of models out of the box.

Status: Internal playtesting.

Download coming soon.

Discord Bot

A discord bot that is capable of understanding and participating in conversation, even conversations with multiple people.

It integrates with discord voice and uses an LLM to generate a response that is then run through a text-to-speech model to make it "talk".

The "voice" of the model can be customized.

Status: Early prototyping.
